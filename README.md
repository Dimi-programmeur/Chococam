# 🍫 CHOCOCAM - Passion du goût

Site vitrine moderne pour une marque chocolatière africaine.  
Présentation des produits, chiffres clés et mise en avant du savoir-faire.

## 📁 Aperçu

Le site est composé d’une page unique avec :
- Une en-tête (navigation + bouton Contact)
- Une section héro avec appel à l’action
- Une section "Meilleur chocolatier"
- Un bloc statistiques (3 millions de consommateurs)
- Une galerie d’illustrations produits
- Un formulaire d’envoi stylisé
- Un pied de page avec le logo

## 🎨 Technologies utilisées

- **HTML5** : structure sémantique
- **CSS3** :
  - Variables CSS (`:root`) pour les couleurs, polices et tailles
  - Flexbox pour la mise en page
  - Positions relative / absolute pour les superpositions d’images
  - Media queries pour le responsive (tablette & mobile)

## 📱 Responsive

Deux breakpoints sont gérés :

| Écran               | Comportement |
|--------------------|--------------|
| 600px – 950px      | Navigation adaptée, masquage de certains blocs |
| < 600px            | Menu masqué, images masquées, centrage du contenu |
